GMM

To estimate the unknown parameter vector $ _{K} $

Suppose there exist moment function \(\mathop{m_t(β)}\limits_{l\times1}\) (assume \(l\geq K\)) and \(E[m_t(\beta^o)]=0\) for unknown true parameter \(\beta^o\)

We construct the sample analog as \(\mathop{\hat m(β)}\limits_{l\times 1}=\frac{1}{n}\sum_{t=1}^n \mathop{m_t(β)}\limits_{l\times 1}\)

Then the GMM estimator is \[ \hat \beta =\arg \min_{\beta \in \Theta} \hat m(\beta) ' \hat W^{-1} \hat m(\beta) \] Remark Moment function may be nonlinear, thus GMM estimator may not have a closed form. But there is a special case when the moment function has the form \(m_t(β) = \mathop{Z_t}\limits_{l\times 1} (Y_t-X_t'\beta)\). Then, GMM estimator has the form like (Proof Required !) \[ \hat \beta = (X'Z\hat W^{-1}Z'X)^{-1}X'Z\hat W^{-1}Z'Y \] 2SLS estimator and OLS estimator is the special case of this linear IV estimator.

8.3 Consistency

We introduce 4 assumptions

  • Assumption 8.1 [Compactness]: Parameter space \(\Theta\) is compact set.

  • Assumption 8.2 [Uniform convergence]:

      1. moment function \(m_t(\beta)\) is measurable function for each \(β\in \Theta\), and given each \(t\), \(m_t(β)\) is continuous in \(β \in \Theta\).
      1. \(\bigl\{m_t(\beta)\bigr\}\) is stationary ergodic
      1. \(\hat m(β)\) convergences uniformly over \(Θ\) to \(m(β)\equiv E(m_t(β))\), i.e. \[ \sup_{\beta\in\Theta}|| \hat m(\beta) - m(\beta) || \xrightarrow{p}0 \]
      1. \(m(β)\) is continuous in \(β \in \Theta\)

  • Assumption 8.3 [Specification]: exist unique true parameter \(β^o\in\Theta\) such that \(m(β^o)=0\)

  • Assumption 8.4 [Weighting matrix]: ${ll} {ll} $ where \(W\) is nonstochastic, symmetric, finite and nonsingular.

Under Assumption 8.1 to 8.4, we can prove (not required) \[ \hat \beta \xrightarrow{p} \beta^o \]

8.4 Asymptotic Normality

We further introduce two assumptions

  • Assumption 8.5 [Interirorness]: \(β^o \in int(Θ)\)

  • Assumption 8.6 [CLT]:

      1. for each \(t\), \(m_t(β)\) is continuously differentiable with respect to \(β \in Θ\) w/p 1
      1. as \(n → \infty\)​​, \[ \sqrt{n} \hat m(\beta^o)\equiv\frac{1}{\sqrt{n}} \sum_{t=1}n m_t(\beta^o) \xrightarrow{d} N(0,V_o) \text{\quad where } V_o \equiv avar(\sqrt{n} \hat m(\beta^o)) \]
      1. \(\left\{\frac{\partial}{\partial\beta}m_t(\beta)\right\}\) obey UWLLN, i.e. \[ \sup_{\beta\in\Theta}\biggl|\biggl| \frac{1}{n}\sum_{t=1}^n \frac{\partial m_t(\beta)}{\partial \beta} - D(\beta) \biggl|\biggl| \xrightarrow{p}0 \\ \text{where } D(\beta)\equiv E\biggl(\frac{\partial m_t(\beta)}{\partial \beta} \biggr) = \frac{\partial E[m_t(\beta)]}{\partial \beta}=\frac{\partial m(\beta)}{\partial \beta} \]

Under Assumption 8.1 to 8.6, we have (Proof Required !) \[ \sqrt{n} (\hat\beta - \beta^o ) \xrightarrow{d} N(0,\Omega) \\ \text{where\quad } \Omega = (D_o'W^{-1}D_o)^{-1}D_o'W^{-1}V_oW^{-1}D_o(D_o'W^{-1}D_o)^{-1} \text{ and } D_o \equiv D(\beta^o) \]

8.5 Asymptotic Efficiency

Given moment function \(m_t(β)\), we want to choose the optimal \(\hat W\) such that GMM estimator has minium variance.

Conclusion is that the optimal choice is (Proof Required !) \[ W = V_0 \equiv avar[\sqrt{n}\hat m(\beta^o)] \] and corresponding variance is \[ \Omega_o = (D_o' V_o^{-1} D_o)^{-1} \] Intuition is that, we assign smaller weights to the moments which have larger sampling variation

Remark: \(\hatβ_{2SLS}\) can be GMM estimator by choosing \(\hat W = Z'Z\). Further, $_{2SLS} $ can be optimal GMM estimator under MDS and HMK. [proof required !]

But in practice, \(V_o\) is not available, so we can use its asymptotic estimator \(\hat W = \tilde V, \text{ where } \tilde V \xrightarrow{p}V_o\). So, we have the following two steps GMM:

  • Step 1: Find a consistent estimator $β $​ by \[ \tilde \beta = \arg\max_{\beta \in \Theta}\hat m(\beta)'\hat W^{-1}\hat m(\beta), \text{\quad where for convenience } \hat W = I \] and we use \(\tilde\beta\) to construct a asymptotic estimtor \(\tilde V \xrightarrow{p} V_o\). Construction depends on MDS:

    • Case I :\(\left\{m_t(β^o)\right\}\) is MDS

      In this case, \(V_o\equiv avar[\sqrt{n}\hat m(β^o)]=E[m_t(β^o)m_t(β^o)']\)

      we construct \(\tilde V\) as follows (Consistency Proof Required !) \[ \tilde V = \frac{1}{n}\sum_{t=1}^{n}m_t(\tilde β)m_t(\tilde β)' \]

    • Case II :\(\left\{m_t(β^o)\right\}\) is non-MDS

      in this case, \(V_o=\mathop{\sum}\limits_{j=-\infty}^{∞}Γ(j)\) where \(Γ(j)=cov[m_t(β_o),m_{t-j}(\beta_o)]=E[m_t(β_o)m_{t-j}(\beta_o)]\)

      we construct \(\tilde V\) as follows \[ \tilde V = \sum_{j=1-n}^{n-1}k(j/p)\tilde \Gamma(j) \text{ where } \tilde \Gamma(j)=\frac{1}{n}\sum_{t=j+1}^n m_t(\tilde \beta)m_{t-j}(\tilde\beta)' \]

  • Step 2: Choose \(\hat W = \tilde V\), then we can find optimal estimator \(\hatβ\): \[ \hat\beta = \arg\min_{\beta\in\Theta} \hat m(\beta)'\tilde V^{-1}\hat m(\beta) \] and \[ \sqrt{n}(\hat\beta -\beta^o) \xrightarrow{d}N(0,\Omega_o) \text{ where }\Omega_o=(D_o'V_o^{-1}D_o)^{-1} \]

8.8 Hypothesis Test

Consider testing null hypothesis \[ \textbf{H}_0: R(\beta^o)=r \] where \(R(\cdot)\) is a \(J\times 1\) continuously differentiable function, which allows for nonlinear restriction. And we want to relate \(R(\hat β)-r\) with \(\sqrt{n}(\hat β-β^o)\).

By Taylor expansion we have \(\mathop{R(\hat β)}\limits_{J\times 1}=\mathop{R(β^o)}\limits_{J\times 1}+\mathop{\frac{dR(\bar \beta)}{dβ}}\limits_{J\times K}\mathop{(\hat β - β^o)}\limits_{K\times1}\) where \(\barβ=λ\hatβ+(1-λ)β^o\).

Thus $$ \[\begin{align} \sqrt{n}[R(\hat\beta)-r] &= \sqrt{n}\mathop{[R(\beta^o)-r]}\limits_{^{H_0} = 0}+\frac{dR(\bar \beta)}{dβ}\mathop{\sqrt{n}(\hat\beta-\beta^o)}\limits_{\xrightarrow{d}N(0,\Omega_o)} \\ &\xrightarrow{H_0}{N(0,\frac{dR(\beta^o)}{dβ}\Omega_o\frac{dR(\beta^o)}{dβ'})} \end{align}\] \[ We can construct the T-test and Wald test statistic \] \[\begin{align} \text{when } J=1: \quad& T \equiv \frac{\sqrt{n}[R(\hat\beta)-r]}{\sqrt{\frac{dR(\hat \beta)}{dβ}\hat\Omega_o\frac{dR(\hat \beta)}{dβ'}}} \xrightarrow{d} N(0,1) \\ \text{when } J\geq1: \quad & W \equiv n [R(\hat\beta)-r]'\biggl[\frac{dR(\hat \beta)}{dβ}\hat\Omega_o\frac{dR(\hat \beta)}{dβ'}\biggr]^{-1}[R(\hat\beta)-r] \xrightarrow{d} \chi^2_J \end{align}\] $$

8.9 Overidentification Test

For the null hypothetis test \[ \text{H}_0: E[m_t(\beta^o)]=0 \text{\quad for some unknown }\beta^o \] Consider the sample analog \(\hat m(\hat \beta)=\frac{1}{n}\sum_{t=1}^nm_t(\hatβ)\) and see whether it is significantly different from 0.

So, we have the following so called \(J\) test statistic [proof not required] \[ J \equiv n \cdot \hat m(\beta)'\tilde V^{-1}\hat m(\beta) \xrightarrow{d}\chi^2_{l-K} \]

  • Remark 1: \(J\) test statistic only valid when \(l\geq K\). (notice the degree of freedom is \(l-K\))
  • Remark 2: \(J\) test statistic is actually "n times the value of step 2 GMM objective function".
  • Remark 3: we prefer a smaller \(J\), since the null hypothesis is IV is valid.
